C19Detect: APP para diagnosticar covid-19
Diciembre, 2021

Formulario Web, desarrollado con Vue y el lenguaje Go, para predecir la probabilidad de estar infectado de Covid-19 basado en data historica recopilada por el gobierno del Perú.

Para ello se uso el algoritmo Random Forest, Docker y Maquins Virtuales de Azure

El formulario consiste de marcar checkbox para saber si el usuario tiene sintomas como:

  • Tos
  • Dolor de garganta
  • Dolor muscular
Al dar click en el boton "Diagnosticar", la aplicación dirá si el usuario seria negativo o positivo de covid-19
